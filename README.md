# TimeClockFX

A simple time clock app with scalaFX.

## Timeline
- Updated 22.12.13
- Created 22.12.13

## Requirements Specification

- Come/Go -function
- Forgot for come/go
- Data storage in a file
- Displaying of the last 10 days
- App resizes in windows desktop DPI scale
